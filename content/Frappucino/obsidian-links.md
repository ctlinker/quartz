## Obsidian: Using Links

### Link Aliasing

In Obsidian, One can alias a link using the pipe | syntax inside \[\[ ... ]]

#### **Basic alias**

\[\[Compile-Time Type Generation|Zig compile-time types]]

This displays:

> Zig compile-time types

but still links to the note:

> Compile-Time Type Generation

---

#### Alias for a heading inside a note

\[\[My Note#Introduction|Intro section]]
